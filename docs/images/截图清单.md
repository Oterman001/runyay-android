# 截图清单

本文件列出《跑步运动助手 软件使用说明书》中所有截图的拍摄要求。

截图拍摄完成后，将文件按以下命名保存至 `docs/images/` 目录，即可自动嵌入说明书文档。

---

## 拍摄通用要求

- 设备：Android 手机（与申请软件一致）
- 分辨率：截图清晰可辨，不低于 1080px 宽度
- 内容完整：截取完整软件界面（含状态栏到导航栏），**不可裁切部分界面**
- 语言：界面语言为中文
- 数据：若界面需要有真实数据展示（如记录列表、统计图表），需使用有真实跑步数据的账号截图
