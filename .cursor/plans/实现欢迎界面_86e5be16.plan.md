# 实现欢迎界面

## 功能概述

参考iOS的 `GuideWellcomePage`，实现Android版本的欢迎界面，作为应用的启动页面。

## iOS参考界面分析

### 布局结构

```
┌─────────────────────────────┐
│  [开发者设置按钮] (仅DEBUG)  │
│                             │
│         (Spacer)            │
│                             │
│      [Logo图片]             │
│       run-duck              │
│      150x150                │
│                             │
│        "跑鸭"               │
│      (标题/粗体)            │
│                             │
│  "一款简约而不简单的        │
│   跑步数据分析应用。"       │
│      (副标题)               │
│                             │
│         (Spacer)            │
│                             │
│     [注册按钮] (蓝色填充)   │
│     [登录按钮] (蓝色边框)   │
│   [访客模式] (灰色文本)     │
│                             │
└─────────────────────────────┘
```

### 关键元素

1. **Logo图片**: `run-duck` (150x150, 圆角12)
2. **标题**: "跑鸭" (title字体, semibold)
3. **副标题**: "一款简约而不简单的跑步数据分析应用。" (subheadline)
4. **注册按钮**: 蓝色填充背景，白色文字
5. **登录按钮**: 透明背景，蓝色边框，蓝色文字
6. **访客模式**: 灰色文字，小字号（Android不需要）
7. **背景**: 渐变色背景 (`Color.backgroudGradient`)

### 2. 创建欢迎页面UI

创建文件 `rundemo/src/main/java/com/oterman/rundemo/presentation/feature/welcome/WelcomeScreen.kt`：

```kotlin
@Composable
fun WelcomeScreen(
    onNavigateToRegister: () -> Unit,
    onNavigateToLogin: () -> Unit
) {
    Box(
        modifier = Modifier
            .fillMaxSize()
            .background(
                brush = Brush.linearGradient(
                    colors = listOf(
                        Color(0xFF4A90E2),
                        Color(0xFF7B68EE)
                    )
                )
            )
    ) {
        Column(
            modifier = Modifier
                .fillMaxSize()
                .padding(horizontal = 40.dp),
            horizontalAlignment = Alignment.CenterHorizontally
        ) {
            Spacer(modifier = Modifier.weight(1f))
            
            // Logo图片
            Image(
                painter = painterResource(id = R.drawable.run_duck),
                contentDescription = "Logo",
                modifier = Modifier
                    .size(150.dp)
                    .clip(RoundedCornerShape(12.dp))
            )
            
            Spacer(modifier = Modifier.height(16.dp))
            
            // 标题
            Text(
                text = "跑鸭",
                style = MaterialTheme.typography.headlineLarge,
                fontWeight = FontWeight.SemiBold,
                color = Color.White
            )
            
            Spacer(modifier = Modifier.height(16.dp))
            
            // 副标题
            Text(
                text = "一款简约而不简单的跑步数据分析应用。",
                style = MaterialTheme.typography.bodyMedium,
                color = Color.White.copy(alpha = 0.9f),
                textAlign = TextAlign.Center
            )
            
            Spacer(modifier = Modifier.weight(1f))
            
            // 按钮区域
            ButtonSection(
                onNavigateToRegister = onNavigateToRegister,
                onNavigateToLogin = onNavigateToLogin
            )
            
            Spacer(modifier = Modifier.height(40.dp))
        }
    }
}

@Composable
private fun ButtonSection(
    onNavigateToRegister: () -> Unit,
    onNavigateToLogin: () -> Unit
) {
    Column(
        modifier = Modifier.fillMaxWidth(),
        verticalArrangement = Arrangement.spacedBy(16.dp)
    ) {
        // 注册按钮（填充背景）
        Button(
            onClick = onNavigateToRegister,
            modifier = Modifier
                .fillMaxWidth()
                .height(56.dp),
            colors = ButtonDefaults.buttonColors(
                containerColor = Color(0xFF4A90E2)
            ),
            shape = RoundedCornerShape(12.dp)
        ) {
            Text(
                text = "注册",
                style = MaterialTheme.typography.titleMedium,
                fontWeight = FontWeight.Medium
            )
        }
        
        // 登录按钮（边框样式）
        OutlinedButton(
            onClick = onNavigateToLogin,
            modifier = Modifier
                .fillMaxWidth()
                .height(56.dp),
            border = BorderStroke(2.dp, Color(0xFF4A90E2)),
            shape = RoundedCornerShape(12.dp)
        ) {
            Text(
                text = "登录",
                style = MaterialTheme.typography.titleMedium,
                fontWeight = FontWeight.Medium,
                color = Color.White
            )
        }
    }
}
```

### 3. 更新导航图

更新 [NavGraph.kt](rundemo/src/main/java/com/oterman/rundemo/presentation/navigation/NavGraph.kt)：

- 将 `startDestination` 改为 `Screen.Welcome.route`
- 添加欢迎页面的路由定义
```kotlin
@Composable
fun AppNavGraph(
    navController: NavHostController,
    startDestination: String = Screen.Welcome.route // 修改启动页
) {
    NavHost(
        navController = navController,
        startDestination = startDestination
    ) {
        // 欢迎页面（新增）
        composable(Screen.Welcome.route) {
            WelcomeScreen(
                onNavigateToRegister = {
                    navController.navigate(Screen.Register.route)
                },
                onNavigateToLogin = {
                    navController.navigate(Screen.Login.route)
                }
            )
        }
        
        // 登录页面
        composable(Screen.Login.route) {
            LoginScreen(
                onLoginSuccess = {
                    navController.navigate(Screen.Home.route) {
                        popUpTo(Screen.Welcome.route) { inclusive = true }
                    }
                },
                onNavigateToRegister = {
                    navController.navigate(Screen.Register.route)
                },
                onNavigateToForgotPassword = {
                    navController.navigate(Screen.ForgotPassword.route)
                }
            )
        }
        
        // ... 其他路由保持不变
    }
}
```


### 4. 添加Logo资源

需要在 `rundemo/src/main/res/drawable/` 目录下添加 `run_duck.png` 或 `run_duck.xml` Logo图片。

可以：

1. 从iOS项目的Assets中导出图片
2. 或者使用占位符图标（如Material Icon的运动图标）

### 5. 主题颜色配置（可选）

如果需要统一主题色，可以在 `Theme.kt` 或 `Color.kt` 中定义：

```kotlin
// 主题蓝色
val ThemeBlue = Color(0xFF4A90E2)
val ThemePurple = Color(0xFF7B68EE)

// 渐变背景
fun backgroundGradientBrush() = Brush.linearGradient(
    colors = listOf(ThemeBlue, ThemePurple)
)
```

## 实现差异说明

### 不实现的功能

1. **访客模式** - 根据用户要求，Android版本不需要访客模式入口
2. **开发者设置** - 调试功能，Android可以通过其他方式实现
3. **区域检测** - iOS的区域功能检测逻辑，Android暂不需要

### 简化的功能

1. **条件渲染** - iOS有多种条件判断（区域、功能开关），Android直接显示注册/登录按钮
2. **功能开关** - iOS使用 `FeatureUtils.isFeatureEnabled`，Android直接显示功能

## 设计规范

### 颜色

- 主色调：蓝色 `#4A90E2`
- 渐变：蓝色到紫色 `#4A90E2` → `#7B68EE`
- 文字：白色

### 间距

- 水平边距：40dp
- 按钮间距：16dp
- 按钮高度：56dp
- Logo尺寸：150dp x 150dp
- 底部间距：40dp

### 圆角

- Logo圆角：12dp
- 按钮圆角：12dp

### 字体

- 标题：headlineLarge + SemiBold
- 副标题：bodyMedium
- 按钮：titleMedium + Medium
